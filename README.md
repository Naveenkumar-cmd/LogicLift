# Zoho Consulting Agency Website

This repository contains a single-page static website for a Zoho consulting agency.
The website is built using HTML, CSS, and JavaScript, all written inside a single index.html file. It uses a vertical scrolling layout with clearly defined sections.

# Sections Included

- Home

- About Us

- Services

- Contact Us

# Features

- Single-page layout

- Vertical section scrolling

- Responsive design

- No backend required

- Contact form integrated using FormSubmit to send inquiries to email

# Tech Stack

- HTML

- CSS

- JavaScript

- FormSubmit (third-party form service)

# Project Structure
- /
- └── index.html

# How to Use

- Clone the repository

- Open index.html in any modern browser

- Update content, styles, or email address as needed

# Notes

- FormSubmit requires email verification on first use

- Suitable for deployment on GitHub Pages, Netlify, or any static hosting service
